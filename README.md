# rasa-chatbot
RASA Chatbot

# Installation

## Virtual Environment

```
python3 -m venv --system-site-packages ./venv
```

### Activate the Virtual Environment

```
source ./venv/bin/activate
```

## Install Rasa Open Source

First make sure your `pip` version is up to date:

```
pip install -U pip
```

To install Rasa Open Source:

```
pip install rasa
```

# Train a Model

```
rasa train
```
